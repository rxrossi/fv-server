# Multi tenant backend for rxmanager

## Setting it up

### MongoDB
```shell
export MONGODB_URI="mongodb://host/dbname"
```

### JWT_SECRET
```shell
export JWT_SECRET="someRandomString"
```

### PORT
```shell
export PORT="5001"
```

### Installing dependencies
```shell
npm install
```

### Running it
```shell
npm start
```

### Running tests

```shell
npm run test:unit
```

or

```shell
npm run test:integration
```
